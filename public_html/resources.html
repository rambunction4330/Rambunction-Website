Kit Of Parts

The Kit of Parts (KOP) is a kit given to every team each year. This kit provides some materials and/or program memberships for the team to use if they so choose, as well as game pieces like FUEL and GEARS. You can find more about the KOP and item checklists on the FIRST website.

Game Manual

The Game Manual includes details on field measurements, points, and rules for the robot and game play. It is vital that every team member read and understand the rules, which you can find here.

Safety Manual

FIRST also provides a Safety Manual so that teams can understand the importance of safety and how to stay safe while building a robot. It is important to go over this manual if you have any questions on whether something is safe or not. You may also ask the team's Safety Captain to assist you, or you can find the manual here.
